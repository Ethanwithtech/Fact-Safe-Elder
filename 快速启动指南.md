# 🚀 AI守护系统 - 快速启动指南

## 📍 方式一：一键启动（推荐）

### Windows系统：

打开**命令提示符**或**PowerShell**，执行：

```powershell
cd D:\Projects\Fact-Safe-Elder
python start_production.py
```

**这个脚本会自动完成所有启动步骤！**

---

## 📍 方式二：分步启动（更稳定）

### 步骤1：检查依赖包

```bash
pip install fastapi uvicorn scikit-learn numpy joblib requests
```

### 步骤2：启动后端服务

**打开第一个终端窗口**，运行：

```bash
cd D:\Projects\Fact-Safe-Elder
python working_backend.py
```

你应该会看到：
```
[OK] AI依赖包加载成功
[OK] AI模型初始化完成
INFO:     Started server process
INFO:     Uvicorn running on http://0.0.0.0:8008
```

### 步骤3：启动前端服务

**保持后端运行，打开第二个终端窗口**：

```bash
cd D:\Projects\Fact-Safe-Elder
python -m http.server 3000
```

或者，如果你想使用专业版前端：

```python
# 创建简单的前端服务器
python -c "from http.server import HTTPServer, SimpleHTTPRequestHandler; import os; os.chdir('D:/Projects/Fact-Safe-Elder'); server = HTTPServer(('', 3000), SimpleHTTPRequestHandler); print('前端服务器运行在 http://localhost:3000'); server.serve_forever()"
```

### 步骤4：访问系统

打开浏览器，访问：
- **主界面**: http://localhost:3000/premium_frontend.html
- **后端API**: http://localhost:8008
- **API文档**: http://localhost:8008/docs
- **健康检查**: http://localhost:8008/health

## 🔧 常见问题排查

### ❌ 问题1：端口被占用

```bash
# 检查端口占用
netstat -ano | findstr ":8008"
netstat -ano | findstr ":3000"

# 清理端口（如果需要）
python kill_ports.py
```

### ❌ 问题2：依赖包缺失

```bash
# 安装所有必需的包
pip install -r backend/requirements.txt
```

### ❌ 问题3：无法访问网页

1. 确保后端服务已启动（检查终端输出）
2. 检查防火墙设置
3. 尝试直接访问 http://127.0.0.1:8008/health

### ❌ 问题4：AI检测显示错误

1. 检查模型文件是否存在：`models/trained/simple_ai_model.joblib`
2. 如果没有，运行训练脚本：
   ```bash
   python simple_ai_trainer.py
   ```
