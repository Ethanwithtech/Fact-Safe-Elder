## 🎯 方法一：Vercel 拖拽部署（推荐）⭐⭐⭐⭐⭐

### 第 1 步：准备文件夹

✅ **已经准备好了**！文件夹位置：
```
D:\Projects\Fact-Safe-Elder\keto-quiz-deploy
```

包含文件：
- ✅ `keto-mom-interactive-quiz.html` - 问卷主文件
- ✅ `index.html` - 首页跳转
- ✅ `vercel.json` - 配置文件

### 第 2 步：访问 Vercel 控制台

1. 打开浏览器，访问：https://vercel.com/dashboard
2. 用您刚才登录的账号（ethanwithtech）登录

### 第 3 步：创建新项目

1. 点击右上角 **"Add New..."** 按钮
2. 选择 **"Project"**

### 第 4 步：导入项目

有两种方式，**选择方式 B 更简单**：

#### 方式 A - 通过 GitHub（推荐，可自动更新）

1. 点击 **"Import Git Repository"**
2. 如果看到 `Fact-Safe-Elder` 仓库，点击 **"Import"**
3. **重要**：在 "Root Directory" 设置中：
   - 点击 **"Edit"**
   - 选择 `keto-quiz-deploy` 文件夹
4. 点击 **"Deploy"**

#### 方式 B - 直接拖拽（最简单，3秒部署）⭐

1. 在项目列表页面找到 **"Deploy without Git"** 或类似选项
2. 或者直接访问：https://vercel.com/new
3. 找到 **"Deploy from a folder"** 或拖拽区域
4. **打开文件资源管理器**，找到：
   ```
   D:\Projects\Fact-Safe-Elder\keto-quiz-deploy
   ```
5. **直接拖拽整个 `keto-quiz-deploy` 文件夹**到浏览器窗口
6. Vercel 会自动上传和部署！

### 第 5 步：等待部署完成

- ⏱️ 通常需要 30 秒 - 1 分钟
- 看到 "Congratulations!" 或类似成功消息

### 第 6 步：获取网址并测试

部署成功后会显示网址，格式类似：
```
https://keto-quiz-deploy.vercel.app
或
https://keto-quiz-deploy-xxx.vercel.app
```

**立即测试**：
1. 点击网址
2. 会自动跳转到问卷页面
3. 测试所有功能是否正常
