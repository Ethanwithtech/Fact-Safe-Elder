## 🎮 使用方法

### 🚀 启动系统
```bash
python start_production.py
```

### 🌐 访问系统
- 前端界面: http://localhost:3000
- 后端API: http://localhost:8008
- API文档: http://localhost:8008/docs

### 📹 视频检测
1. **方法一**: 拖拽视频文件到上传区域
2. **方法二**: 点击"选择视频文件"按钮上传
3. **可选**: 输入视频文字描述提高准确度
4. **查看结果**: 系统自动显示检测结果和建议

### 🤖 AI训练
1. 配置训练参数（轮数、批次、学习率）
2. 选择模型类型（SVM推荐）
3. 点击"开始训练AI模型"
4. 等待训练完成（2-5分钟）
5. 新模型自动部署
