## ⚠️ 发现的问题

### 问题 1：学校网络的 AP 隔离 ⭐⭐⭐（最可能的原因）
**学校/企业 Wi-Fi 通常启用了 AP 隔离（Client Isolation）**，这会阻止同一网络下的设备互相访问，即使在同一个 Wi-Fi 下也无法互通。

**这是安全措施，用于保护学生/员工的设备不被其他人访问。**

### 问题 2：VPN 可能干扰
NordVPN 正在运行，可能会影响局域网流量路由。

### 问题 3：防火墙（已经讨论过）
Windows 防火墙可能阻止端口 8080。

---

## 🎯 解决方案（按优先级）

### 方案 A：使用手机热点（推荐，最简单）⭐⭐⭐⭐⭐

**这是最可靠的方法！**

1. **用手机开启热点**
   - 打开手机设置
   - 启用"个人热点"或"便携式热点"
   - 设置热点名称和密码

2. **电脑连接到手机热点**
   - 断开 HKBU Wi-Fi
   - 连接到手机热点

3. **其他设备也连接到同一个手机热点**

4. **运行服务器**
   ```powershell
   python serve_quiz.py
   ```
   会显示新的 IP 地址（通常是 192.168.x.x）

5. **其他设备用新的 IP 访问**

**优点**：
- ✅ 100% 可行，没有 AP 隔离
- ✅ 不受学校网络限制
- ✅ 简单快速

---

### 方案 B：暂时关闭 VPN 并配置防火墙

1. **关闭 NordVPN**
   - 退出 NordVPN 应用
   - 或者在 NordVPN 中断开连接

2. **配置防火墙**（以管理员身份运行 PowerShell）：
   ```powershell
   netsh advfirewall firewall add rule name="Python HTTP Server 8080" dir=in action=allow protocol=TCP localport=8080
   ```

3. **重启服务器**
   ```powershell
   python serve_quiz.py
   ```

4. **测试访问**

**注意**：如果学校网络有 AP 隔离，这个方法也不会成功。

---

## ❓ 常见问题

### Q: 为什么学校网络不行？
A: 学校/企业网络为了安全，会启用 AP 隔离，阻止设备间通信。这是正常的安全措施。

### Q: 手机热点会消耗很多流量吗？
A: 不会。这个问卷是纯 HTML，只有 1-2MB，而且只需要加载一次。

### Q: 可以用 USB 连接吗？
A: 可以，但配置复杂。手机热点是最简单的方法。

### Q: VPN 一定要关吗？
A: 不一定，但 VPN 可能会干扰局域网流量，建议测试时先关闭。
