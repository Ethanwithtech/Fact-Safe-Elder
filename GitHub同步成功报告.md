### 2. 配置GitHub Actions CI/CD
```yaml
# .github/workflows/ci.yml (可选)
name: AI守护系统 CI/CD
on: [push, pull_request]

    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: 测试前端
        run: cd frontend && npm test
```
